2. Performance Monitering
=> Core Web Vitals
  => LCP(Largest contentful paint) => 2.5s to 4s => Loading Time
  => FID(First Input deplay) => 100ms - 300ms => Interactivity
  => CLS(Cumulative Layout shift) => 0.1s - 0.25s => Visual Stability
  => INP(Interraction to next paint) => 200ms - 500ms

=> Browser centric matrics
  => Time to first byte(TTFB)
  => Network Request
  => DNS Resolution
  => Connection time
  => DOM Content Load
  => Page Load

=> User centric matric
